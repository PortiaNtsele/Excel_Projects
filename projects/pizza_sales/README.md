# Pizza Sales Analysis

## Overview
This project analyzes a pizza shop’s sales data to understand revenue trends and product performance. The analysis includes data cleaning, merging multiple datasets, and creating visualizations to summarize insights.

## Dataset
All raw data files are located in the `dataset/` folder:

- `order_details.csv` — individual pizza orders
- `orders.csv` — order metadata including dates and times
- `pizza_types.csv` — types of pizzas and categories
- `pizzas.csv` — pizza IDs, sizes, and prices

## Analysis
All analysis files are located in the `analysis/` folder:

- `Merged_Dataset.xlsx` — cleaned and merged dataset used for analysis
- `Pivot_Revenue_Category.png` — revenue by pizza category
- `Pivot_Revenue_Day.png` — revenue by day of the week
- `Pivot_Revenue_Month.png` — revenue by month

## Skills Demonstrated
- Data cleaning and merging multiple datasets in Excel
- Creating pivot tables and pivot charts
- Summarizing revenue trends by category, day, and month
- Presenting analysis in a structured and professional format
